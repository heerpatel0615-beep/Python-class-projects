# Interactive Personal Data Collector

## Overview

This is a beginner-friendly Python project created using Jupyter Notebook.
The program collects user information and demonstrates basic Python concepts.

## Concepts Used

* print()
* input()
* datatypes
* variables
* operators
* type casting
* type()
* id()

## Features

* Collects user data
* Shows data types
* Calculates birth year
* Demonstrates type casting

## Technologies Used

* Python
* Jupyter Notebook
* GitHub
